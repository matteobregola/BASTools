Simulation

Ref Feature Name Description
E1 Cleanup The tool permits performing a complete cleanup of the system after the execution of the simulation, restoring its original status
E2 Simulation Live Info The tool provides the user with real-time information about the simulation it is exe- cuting
E3 Pause The tool permits pausing the simulation during its execution
E4 Override The tool permits the user to pause the simulation and change its configurations
E5 Interrupt The tool permits the user to pause the simulation during its execution
E6 Schedule The tool can automatically execute simulations without user interaction and permits the user to schedule them
E7 Import Simulation The tool permits importing a simulation's plan from a file

The features in this section analyse the overall capabilities of managing simulations before, during, and after their execution.
