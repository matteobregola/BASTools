Tool finder

This section is a

Beta

, the idea is to provide the user a list of features that it has to choose based on their requirements, then evaluate which tool satisfies these features and return them. As for now there are two possible form that can be used to assist the choice:

  • Default

    -> Contains a subset of the framework's features that is faster to use and get a result.

  • Complete

    -> Contains all the framework's features, and it can be used if the user is interested in specific features.
