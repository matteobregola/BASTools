Other

Ref Feature Name Description
G1 Plugins The tool permits to integrate plugins developed by the users.
G2 CTI Updates The tool informs the users about new vulnerabilities and CTI   reports, indicating also if the tool developed some new activities related to   test security against them.
G3 Documentation The tool offers documentation accessible to the users.
G4 Interactive Tutorial The tool offers an interactive tutorial to explain to users   its usage and its capabilities
G5 General Vulnerability Score The tool offers a quantitative evaluation of the overall   system vulnerability based on all performed simulations [1].
G6 General Risk Score The tool offers a quantitative evaluation of the overall   system risk level based on all   performed simulations [1].
G7 General Security Score The tool offers a quantitative evaluation of the overall   system security based on all performed simulations [1]
G8 General Security Trend The tool provides an analysis of the system’s security trends   over time and across simulations [1].
G9 Impact level The tool allows the user to manually specify the impact of   attack activities on the user’s business.
G10 Targets Relevance The tool allows the user to manually specify the relevance of   each asset in the system[1]
G11 Tag Creation The tool allows the user to create tags that can be added to   activities or targets.
G12 Target Customization The tool allows the user to customize the targets of the   system
G13 Manual Targets Grouping The tool permits the create and save groups of targets within   the system[1].
G14 Automatic Targets Grouping The tool provides an automatic grouping of the targets of the   system (e.g.: devides databases from devices and workstations)
G15 Compliance The tool is capable of validating the compliance with one or   more security standards.
G16 Standalone The BAS tool can be bought or used as a standalone software   without integrating other functionalities [1]
G17 Automatic Mitigation The tool is capable of applying mitigations for the discovered   vulnerabilities.
